---
title: Variables de Entorno
description: Lista de variables usadas por el backend y su propósito.
---

El backend carga variables desde `server/.env` (ver `server/core/config.py`).

```ini
# Base de datos
DATABASE_URL=

# Seguridad
JWT_SECRET=
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=60

# Email (recuperación de contraseña / notificaciones)
EMAIL_SENDER=
EMAIL_PASSWORD=

# Spotify
SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
SPOTIFY_REDIRECT_URI=http://127.0.0.1:8000/v1/auth/spotify/callback

# AWS Rekognition
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_REGION=us-east-2
AWS_REKOGNITION_MAX_LABELS=10
AWS_REKOGNITION_MIN_CONFIDENCE=75
AWS_REKOGNITION_SIMILARITY_THRESHOLD=90
```

Sugerencias:

- Usá cuentas/roles mínimos de AWS para evitar sobrepermisos.
- Nunca subas `.env` al repositorio.
- En producción, preferí gestores de secretos.
